Predefinidos

CustomCSSButton ofrece un conjunto de botones predefinidos, de tamaño intermedio, que permite la compatibilidad en distintas resoluciones. Incorpora una animación por defecto obteniendo un botón con un estilo realista y original.

Personalizar

  1. Introducir nombre
  2. Seleccionar tamaño
  3. Activar zona a pintar (fondo, texto, sombra)
  4. Seleccionar el color con la paleta interactiva
  5. Pulsar OK
  6. Comprobar la animación con el botón de muestra
  7. Pulsar icono de visibilidad (ojo)
  8. Pulsar botones para copiar código HTML y CSS al portapapeles
