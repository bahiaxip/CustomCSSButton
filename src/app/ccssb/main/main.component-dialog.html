CustomCSSButton es una aplicación práctica y sencilla que permite, mediante una interfaz gráfica, generar el código HTML y CSS de un botón personalizado.
CustomCSSButton incorpora una paleta interactiva y un botón de previsualización, obteniendo como resultado la edición y visualización de un botón animado con estilo propio en solo unos segundos.

CustomCSSButton ofrece un conjunto de botones predefinidos de tamaño intermedio, compatible en distintas resoluciones con un efecto de animación incorporado por defecto ofreciendo un tipo de botón con un estilo realista y original.

Introducir nombre
Seleccionar tipo
Seleccionar tamaño
Introducir texto
Activar zona a pintar (fondo, texto, sombra)
Seleccionar el color mediante la paleta interactiva y pulsando OK
Accionar icono de visibilidad (icono en forma de ojo)
