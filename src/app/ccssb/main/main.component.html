CustomCSSButton

CustomCSSButton es una aplicación práctica y sencilla que permite, mediante una interfaz gráfica, generar el código HTML y CSS de un botón personalizado.
CustomCSSButton incorpora una paleta interactiva y un botón de previsualización, obteniendo como resultado la edición y visualización de un botón animado con estilo propio en solo unos segundos.

Predefinidos

CustomCSSButton ofrece un conjunto de botones predefinidos de tamaño intermedio, compatible en distintas resoluciones con una animación incorporada por defecto ofreciendo un tipo de botón con un estilo realista y original.

Personalizar

  1. Introducir nombre
  2. Seleccionar tipo
  3. Seleccionar tamaño
  4. Introducir texto
  5. Activar zona a pintar (fondo, texto, sombra)
  6. Seleccionar el color mediante la paleta interactiva pulsando OK
  7. Accionar icono de visibilidad (icono en forma de ojo)
